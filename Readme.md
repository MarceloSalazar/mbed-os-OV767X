# OV767X Library for Mbed OS

Capture images from your OmniVision OV7670 camera in your Mbed OS applications sketches.

This library is based on [Linux Kernel's](https://www.kernel.org) V4L2 driver for OmniVision OV7670 cameras - which was created by Jonathan Corbet.
Additionally, some parts of this library are based on the [Arduino_OV767X](https://github.com/arduino-libraries/Arduino_OV767X/) library published here:

## Add the library to your Mbed OS project

```
mbed add <URL>
```
